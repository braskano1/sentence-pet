# Curriculum Roadmap — Sentence Pet

The course ladder for the sentence-builder game, by CEFR level. Each **course**
is a self-contained "world": ~5–6 units, a final boss, and a reward pet. We ship
courses as **series of short courses per level**, not one mega-course — each
finale is an evolution/collection milestone (the game's retention hook), and
each course recycles the prior course's vocab in new frames (built-in spaced
repetition).

Courses are authored as `.xlsx` workbooks (Course/Units/Items/Bosses sheets) via
the guides in [`authoring/`](authoring/), generated by `scripts/gen-c*.mjs`, and
verified through `parseWorkbookToCourse` + `validateCourse` before import. Import
in admin → **Courses → Import**.

---

## The engine constraint that shapes everything

Drag-build sentences are **slot-filled**. Grading is an **ordered token match**
(`src/domain/placement.ts`) — slot **labels are display scaffold only**, never
checked against a word's real POS. Consequence: **adding a slot label is cheap**
(extend the `PosLabel` union + a chip color + the author guide; no grading
change). That's why each level can unlock new sentence shapes with a tiny code
delta instead of an engine rewrite.

### Slot label evolution

| Label | Holds | Added at |
|---|---|---|
| Subject · Verb · Object | I/run/rice | Pre-A1 C1 (baseline; renamed from Pronoun→Subject) |
| Be · Adjective | am-is-are / happy | Pre-A1 C2 |
| Not · Helper · Question · Place | don't / do / where / in | Pre-A1 slot extension |
| **Adverb** | always/usually/never | **A1 (planned)** |
| **Determiner** | my/your/his/her | **A1 (planned)** |

Display punctuation: `DragDropItem.endPunct` (`.`/`?`) + the xlsx `punct` column
exist; `renderSentence(words, endPunct)` honors it. **Not yet surfaced in the
live drill display** (no production caller) — wire it when question-heavy A1
content ships.

## Pre-A1 — ✅ DONE (branch `slot-label-subject`)

Five courses, ~541 items. Engine caps drag-build to the SVO + be + neg + question
frames; vocabulary stays tiny and recycled (mass/plural nouns to dodge articles;
articles taught via fillblank/matching).

| Course | id | Items / Units | Teaches |
|---|---|---|---|
| C0 ABC & First Words | `pre-a1-c0-abc` | 51 / 5 | alphabet, colors, greetings, classroom (vocab on-ramp; no gates, single final) |
| C1 First Sentences | `pre-a1-first-sentences` | 127 / 6 | SVO present simple, 3rd-person -s |
| C2 I Am, You Are | `pre-a1-c2-i-am` | 123 / 6 | be (am/is/are) + adjectives + this/that + a/an |
| C3 Yes, No, and Not | `pre-a1-c3-yes-no` | 117 / 6 | negatives + yes/no questions + short answers |
| C4 Where? What? How Many? | `pre-a1-c4-where-what` | 123 / 6 | wh-questions + there is/are + prepositions + numbers/age |

Conventions: 6 units (C0=5), ~5 items/lesson, checkpoint = mixed dragdrop = the
**last node group per unit**, 2 gated bosses (after u2 + u4) + 1 final (C0 = 0
gates), `rewardPetDefId` blank for admin to assign, Thai L1 on every item.

## A1 — PLANNED (elementary)

Builds on Pre-A1. Same series shape: ~5 short courses, each a world + final boss
+ reward pet. ~600 items total — **playtest Pre-A1 first**, then scale depth to
what completion-rate / boss-retry data shows.

| # | Course | Teaches | Engine fit |
|---|---|---|---|
| A1-C1 | **Right Now** | present continuous (`I am running`) + can/can't (ability) | ✅ `Subject+Be+Verb`; `I,can,swim` (can→Helper); `Can you swim?` |
| A1-C2 | **My Family** | possessive adjectives (my/your/his/her) + `'s` + object pronouns (me/him/her) + have | needs **Determiner** slot (`my dog is big`) |
| A1-C3 | **Every Day** | adverbs of frequency (always/usually/sometimes/never) + daily routine + time + prepositions of time (in/on/at) | needs **Adverb** slot (`I always eat rice`) |
| A1-C4 | **I Like It** | like/love/hate + -ing, want to, some/any, (basic comparatives) | ✅ gerund rides Object (`I,like,swimming`); `me/him/her` in Object |
| A1-C5 | **Yesterday** | past simple of *be* (was/were) + there was/were | ✅ was/were → Be slot; bridges to A2 |

### A1 engine delta (small, additive — same cheap pattern)
1. Add slot labels **`Adverb`** and **`Determiner`** to `PosLabel` + chip colors + author guide.
2. **Surface `endPunct`** in the drill display (carry-forward from Pre-A1).

Everything else is data-only workbooks (`gen-a1-c*.mjs`).
